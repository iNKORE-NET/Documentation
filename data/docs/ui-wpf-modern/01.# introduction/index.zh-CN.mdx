# 简介

随着 Windows 应用程序开发的不断发展，开发人员一直在寻求使他们的应用程序既美观又现代的解决方案。传统的 Windows Presentation Foundation (WPF) 已经成为许多桌面应用程序的基石，提供了强大的功能和灵活性。然而，随着用户界面设计趋势的变化，尤其是微软引入 Fluent Design 系统之后，开发人员迫切需要一种方法来实现现代设计美学，而无需转向新的框架。

## 这是什么？

iNKORE.UI.WPF.Modern 是一个创新的 Windows Presentation Foundation (WPF) 应用程序 UI 库，旨在实现 WinUI 中的 Fluent Design 系统的时尚和现代美学。此库使开发人员能够在不依赖 WinUI 或 UWP 框架的情况下为他们的 WPF 应用程序增添现代的外观和感觉，且即使在 Windows 7 等旧操作系统上也能完全兼容。

## 主要功能

- **Fluent Design 设计体系**：iNKORE.UI.WPF.Modern 集成了 Fluent Design System 的美学元素，其以强调光线、深度、动效、材质和比例而闻名。我们的库将这些元素带入您的 WPF 应用程序，提供现代且一致的用户体验，视觉上令人印象深刻且功能多样。

- **纯 WPF 实现**：与其他需要额外框架或扩展的库不同，iNKORE.UI.WPF.Modern 完全基于 WPF 构建。这意味着更强的兼容性和更简单的现有 WPF 项目集成。

- **高兼容性**：iNKORE.UI.WPF.Modern 设计为在 Windows 7 及更新版本上完美运行，以确保您的应用程序可以在不妥协设计或功能的情况下覆盖更广泛的受众。

- **无外部依赖**：您不需要安装额外的软件包或管理复杂的依赖关系。iNKORE.UI.WPF.Modern 是完整提供的，简化了您的开发过程并减少了现代 UI 实现通常需要的相关开销。

- **易于集成**：简单的设置和配置过程，使您能够立即开始编辑您的应用程序。

- **灵活且完全可定制的主题**：通过完全适应和定制的主题，定制组件以匹配您的应用程序的独特风格。

:::tip 提醒

为了获取系统主题颜色、配色方案和其他一些设置，本库有一个可选的 WinMD 库引用。在 Windows 10 1809 及更早版本中，主题无法自动检测，将应用默认设置。如果需要，您可以随时进行自定义。

:::

## 为什么选择我们？

选择 iNKORE.UI.WPF.Modern 意味着选择了一个弥合了传统设计与现代设计之间的差距的 UI 库，它在 WPF 生态系统内提供了一种无需过渡到新框架的现代化应用程序的方法，从而显著减少学习曲线和开发时间。

那些将此库集成到项目中的开发人员一致称赞其易用性和对应用程序的显著视觉升级。反馈强调了该库如何帮助提供现代的用户体验，使应用程序在视觉上更具吸引力和吸引力。
